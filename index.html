Conectar na Região Tomix Modded!

Se você estiver em um dispositivo móvel, clique aqui para se conectar ao servidor.

Para usuários de PC, siga estas instruções:

  1. Faça o download do arquivo regioninfo.json aqui.
  2. Aperte Windows + R e cole o seguinte caminho: %userprofile%\AppData\LocalLow\Innersloth\Among Us
  3. Cole o arquivo transferido nessa pasta.
  4. Abra o Among Us.

Lamentamos, mas no Mac é impossível baixar e usar arquivos de região modded para Among Us. Sugerimos usar um PC para esta tarefa.
